首先要回答的问题是何为渲染：

渲染（render）对于浏览器来说，仅代表着一件事，就是将 HTML/CSS 转换为屏幕上像素信息的这一整条完整流水线；对于vue或者react等框架来说，渲染则意味着计算变更后的 UI 应该长什么样（虚拟 DOM），并更新真实 DOM 来匹配这个状态。

```
渲染流水线：
HTML 字符串
 → 解析 HTML → DOM 树
 → 解析 CSS → CSSOM 树
 → DOM + CSSOM → Render 树
 → 布局（Layout）：计算每个元素的位置和大小
 → 绘制（Paint）：生成绘制指令
 → 合成（Composite）：拆层、GPU 合成、输出到屏幕
```
