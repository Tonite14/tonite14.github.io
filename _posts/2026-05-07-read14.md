## 尾调用优化：

JavaScript引擎会在满足条件时重用栈帧以优化内存管理。具体来说，这项优化非常适合尾调用，即外部函数的返回值是一个内部函数的返回值。比如：

```js
function outerFunction() {
  return innerFunction(); // 尾调用
}
```
